# Lucas Coelho — Portfólio

Site pessoal e portfólio de Lucas Coelho, desenvolvedor fullstack com +12 anos de experiência construindo sistemas em produção.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router DOM

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
